# Project: Pomodoro Timer

## Tech Stack
- Single-file HTML/CSS/JS (`index.html`)
- No frameworks, no build tools, no dependencies
- LocalStorage for persistence

## Quality Gates (must ALL pass before committing)
- Open `index.html` in a browser — no console errors
- All buttons respond to clicks
- Timer display updates correctly

## Rules
- Everything lives in ONE file: `index.html` (inline `<style>` and `<script>`)
- NEVER add external dependencies or CDN links
- NEVER use alert() for notifications — use the document title
- Keep CSS minimal and clean — centered layout, readable font sizes
- Use `setInterval` / `clearInterval` for timer logic
- Use semantic HTML elements
